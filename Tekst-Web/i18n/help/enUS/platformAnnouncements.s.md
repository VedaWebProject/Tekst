# Announcements

Announcements are displayed to visitors in a large pop-up as soon as they visit the site. You can use them to communicate important information, such as maintenance work or significant changes to the platform.

You can create a separate announcement for each language. If you want to cover multiple topics in your announcements, you can divide them into different sections.

Visitors can disable the automatic display of announcements either permanently or until the next update to the announcements.
