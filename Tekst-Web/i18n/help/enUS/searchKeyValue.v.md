# Key-Value Search

A key-value search is available in the Advanced Search for resources of the following types:

- Text Annotations
- Location Metadata

This search type allows you to look specifically for particular key-value pairs. The search form provides a select menu with the available keys. For the values, an exact term search among a selection of existing values is possible.

Alternatively, values can be searched using wildcards. To do this, activate the button with the `*` symbol in the corresponding entry. Omissions can be marked with `?` for a single character or `*` for 0–n characters. Unlike full-text searches, both prefix and suffix searches using wildcards are possible here.
