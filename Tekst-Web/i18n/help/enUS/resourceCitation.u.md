# Resource Citation

Sets the citation for this resource.

## Automatic Replacements

You can use the following placeholders, which will be automatically replaced:

- `{{res_url}}`: The URL of the information overview for the respective resource
- `{{curr_date}}`: The current date when displaying the citation (in the local format of the set language)

## Special Case for Line Breaks

Citations can contain line breaks if needed. This can enable combining multiple citations sequentially for merged resources.

If a citation contains at least one line break, a simplified citation without integrated citation URL will be displayed for locations of this resource in the reading view.
