# Schlüssel-Wert-Suche

Eine Schlüssel-Wert-Suche steht in der erweiterten Suche für Ressourcen der folgenden Typen zur Verfügung:

- Text-Annotationen
- Belegstellen-Metadaten

Bei dieser Suche kann gezielt nach bestimmten Schlüssel-Wert-Paaren gesucht werden. Das Suchformular stellt ein Select-Menü mit den verfügbaren Schlüsseln zur Verfügung. Für die Werte ist eine exakte Term-Suche nach einer Auswahl vorhandener Werte möglich.

Alternativ kann nach Werten auch mit Hilfe von Wildcards gesucht werden. Aktivieren Sie hierzu den Button mit dem `*`-Symbol im entsprechenden Eintrag. So können Auslassungen mit `?` für ein einzelnes Zeichen oder `*` für 0-n Zeichen gekennzeichnet werden. Anders als bei den Volltextsuchen sind hier sowohl Präfix- als auch Suffix-Suchen mittels Wildcards möglich.
