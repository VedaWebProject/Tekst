# Neuer Text

Hier können Sie einen neuen Text anlegen. Dafür müssen Sie nur die grundlegendsten Angaben machen. Zunächst wird der Text nicht öffentlich gelistet und ist nur für Administrator\*innen zugänglich. Sie werden im Anschluss zu einer Ansicht weitergeleitet, in der Sie weitere Einstellungen vornehmen können, bevor Sie den Text öffentlich verfügbar machen.

## Ebenen

Hier handelt es sich um die Bezeichnungen für die strukturellen Ebenen, mit denen Sie den Text abbilden möchten. Sie können diese im Nachhinein ändern und auch mehrsprachig ausgestalten. Auch wenn die hier gewählten Bezeichnungen nicht endgültig sind, ist es an dieser Stelle aber ratsam, bereits die Anzahl und die Art der geplanten Ebenen festzulegen, da ein späteres Löschen oder Hinzufügen von Ebenen zum Verlust von strukturellen Informationen führen kann, die nur mit viel Aufwand wiederhergestellt werden können.

Ein simples Beispiel für die Ebenen eines Textes wäre etwa folgendes:

1. Band
2. Kapitel
3. Absatz
4. Satz

Wie genau Sie die Struktur des Textes abbilden, hängt aber ausschließlich von der Natur der Daten ab, die Sie einbinden möchten. Wenn Sie z.B. Annotationsdaten für einzelne Wörter einbinden möchten, dann sollte die "unterste" Sturkturebene mindestens die nächst höhere Ebene von der Wort-Ebene aus sein (etwa "Satz" oder "Absatz"), da es wenig sinnvoll wäre, Wort-spezifische Annotationen im Kontext ganzer Kapitel zu betrachten.
