# Gemeinschaft

In der Gemeinschafts-Ansicht erhalten Sie einen Überblick über alle registrierten Benutzer\*innen der Plattform. Oberhalb der Liste finden Sie eine Suchfunktion, mit der sich die Liste filtern lässt. Ein Klick auf einen Eintrag in der Liste öffnet das entsprechende Nutzer\*innenprofil, wo sich auch Nachrichten an die jeweilige Nutzer\*in senden lassen.
