# Auswahl der Ebene und Belegstelle

Hier können Sie gezielt eine Belegstelle auswählen, für die Sie sich Inhalte anzeigen lassen möchten. Belegstellen können sich auf unterschiedlichen Strukturebenen eines Textes befinden, z.B. kann eine Belegstelle ein Kapitel, ein Absatz oder ein Satz sein.

Unterschiedliche Ressourcen können mit verschiedenen Strukturebenen eines Textes assoziiert sein. Generell werden Inhalte von übergeordneten Belegstellen – also aus hierarchisch höheren Strukturebenen – mit angezeigt. Deshalb ist es in den meisten Fällen wünschenswert, Inhalte auf der untersten Strukturebene anzuzeigen. Falls sie jedoch z.B. kombinierte Inhalte einer höheren Ebene sehen möchten (etwa "alle Sätze aus Kapitel 1"), können Sie hier die Ebene entsprechend auswählen.

Natürlich sind die oben genannten Ebenen-Bezeichnungen nur Beispiele und können je nach Text völlig andere sein. Sie können die verfügbaren Ebenen der Auswahlliste entnehmen.
