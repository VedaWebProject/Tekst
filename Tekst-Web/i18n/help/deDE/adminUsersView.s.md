# Benutzer\*innenverwaltung

Als Administrator\*in haben Sie hier volle Kontrolle über die registrierten Konten und deren Status. Sie können Benutzer\*innenkonten verifizieren, aktivieren, deaktivieren, löschen oder Administrationsrechte vergeben und entziehen.

## Aktionen im Überblick

- Sie können die Liste der Benutzer\*innen nach verschiedenen Kriterien filtern. Öffnen Sie dazu die Filter-Funktion über der Liste.
- Klicken Sie auf einen Namen in der Liste, um zur entsprechenden Profilseite zu gelangen (von hier aus können Sie auch eine Nachricht an die Benutzer\*in senden).
- Klicken Sie auf eine EMail-Adresse, um eine EMail an die Benutzer\*in zu verfassen.
- Klicken Sie auf die drei Punkte am rechten Rand eines Eintrags, um den Status des jeweiligen Kontos zu bearbeiten.

## ⚠️ WARNUNG zum Löschen von Konten

Wenn Sie ein **Benutzer\*innenkonto löschen**, werden alle damit verbundenen Daten unwiderruflich gelöscht. Dies kann nicht rückgängig gemacht werden. Wenn eine Benutzer\*in alleinige Besitzer\*in einer unveröffentlichten Ressource ist und das Konto gelöscht wird, wird die Ressource samt ihrer Inhalte ebenfalls gelöscht. Ist die Ressource veröffentlicht, wird eine zufällige Administrator\*in als neue Besitzer\*in festgelegt.

## ⚠️ WARNUNG zum Vergeben von Administrator\*innenrechten

Wenn Sie einer Benutzer\*in administrative Rechte zuteilen, kann diese\*r Konten, Ressourcen und Inhalte verwalten, bearbeiten und löschen. Bitte tun Sie dies nur, wenn Sie sicher sind, dass die entsprechende Person die Verantwortung hierfür übernehmen kann!
