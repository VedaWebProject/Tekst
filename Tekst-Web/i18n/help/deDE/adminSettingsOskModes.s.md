# Bildschirmtastaturen

Sie Bildschirmtastaturen zur Verfügung stellen, damit Nutzer\*innen mit abweichenden Tastaturlayouts vereinfacht Eingaben machen können, die dem Zeichensatz der Ressourcen dieser Plattform entsprechen.

## Installation von Tastaturlayouts

Bitte entnehmen Sie dem Kapitel "Setup / Customization" des [Tekst-Handbuches](https://vedawebproject.github.io/Tekst/), wie Layouts für Bildschirmtastaturen hinterlegt werden können. Die Dateinamen der Layouts können dann hier als Schlüssel angegeben werden, um das entsprechende Layout zu referenzieren. Der **Name** wird den Nutzer\*innen in der Auswahl der verfügbaren Bildschirmtastaturen angezeigt.

## Schriftarten für Bildschirmtastaturen

Falls eine Bildschirmtastatur einen speziellen Zeichensatz abbildet, der nicht durch die Standardschriftart für Inhalte dargestellt werden kann, können Sie ihr hier gezielt eine Schriftart zuweisen. Bitte beachten Sie, dass diese Schriftart vorher korrekt eingerichtet werden muss (siehe Einstellungen zu "Schriftarten").
