# Ressourcen-Zitation

Legt die Zitationsvorgabe für diese Ressource fest.

## Automatische Ersetzungen

Sie können folgende Platzhalter einsetzen, die dann automatisch ersetzt werden:

- `{{res_url}}`: Die URL der Informationsübersicht der jeweiligen Ressource
- `{{curr_date}}`: Das aktuelle Datum beim Anzeigen der Zitationsvorgabe (im lokalen Format der eingestellten Sprache)

## Besonderheit bei Zeilenumbrüchen

Zitationsvorgaben können wenn nötig Zeilenumbrüche enthalten. Dies kann etwa das kombinieren mehrerer Zitationen nacheinander bei zusammengeführten Ressourcen ermöglichen.

Falls eine Zitationsvorgabe mindestens einen Zeilenumbruch enthält, wird für Belegstellen dieser Ressource in der Leseansicht eine vereinfachte Zitationsvorgabe ohne integrierte Belegstellen-URL angezeigt.
