# Bekanntmachungen

Bekanntmachungen werden den Besucher*innen der Seite in einem großformatigen Pop-Up angezeigt, sobald sie die Seite besuchen. Sie können sie nutzen, um wichtige Informationen zu kommunizieren, z.B. über Wartungsarbeiten oder wichtige Änderungen an der Plattform.

Es kann für jede gewünschte Sprache eine eigene Bekanntmachung erstellt werden. Wenn Sie mehrere Themen in ihren Bekanntmachungen behandeln möchten, können Sie diese in verschiedene Abschnitte unterteilen.

Besucher*innen können das automatische Anzeigen der Bekanntmachungen ganz oder bis zur nächsten Änderung der Bekanntmachungen abschalten.
