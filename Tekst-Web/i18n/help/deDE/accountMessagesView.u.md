# Nachrichten-Übersicht

Hier sehen Sie alle gespeicherten Konversationen zwischen Ihnen und anderen Nutzer\*innen sowie System-Meldungen, die Ihnen automatisch zugestellt wurden. Um eine Nachricht an eine\*n Nutzer\*in zu verschicken, öffnen Sie eine vorhandene Konversation oder besuchen Sie die entsprechende Profilseite und klicken dort auf den Nachrichten-Button.

Ein Klick auf den Löschen-Button (Mülleimer-Symbol) löscht die gesamte Konversation – jedoch nur für Sie. Die andere Person kann die Konversation weiterhin sehen, solange sie von ihr nicht ebenfalls gelöscht wird.

Wenn Sie ihr Konto löschen, werden alle von Ihnen verschickten Nachrichten automatisch ebenfalls gelöscht.
