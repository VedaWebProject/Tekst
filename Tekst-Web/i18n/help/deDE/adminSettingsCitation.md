# Einstellungen für Zitationsvorgaben

## Globales Suffix für Zitationen

Dieser Text wird automatisch an die Zitationsvorgaben aller Ressourcen angehängt. Genau wie in diesen Zitationsvorgaben selbst, können sie auch hier folgende Platzhalter einsetzen, die dann automatisch ersetzt werden:

- `<res-url>`: Die URL der Informations-Übersicht der jeweiligen Ressource
- `<curr-date>`: Das aktuelle Datum beim Anzeigen der Zitationsvorgabe (im lokalen Format der eingestellten Sprache)
